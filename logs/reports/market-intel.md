## 数据驱动的优先级建议

基于以上情报，建议优先解决：

### P0 优先级（必须解决）
1. **首页组件替换**
   - 理由: 首页是用户第一印象，影响留存率
   - 数据支撑: 竞品（试题通、疯狂刷题）都有现代化首页设计
   - 技术方案: 使用 WelcomeBanner, StatsCard, KnowledgeBubble

### P1 优先级（应该解决）
2. **个人中心组件替换**
   - 理由: 个人中心是用户高频访问页面
   - 数据支撑: 用户反馈"统计数据展示不够直观"
   - 技术方案: 使用 ActivityItem, StatsCard

3. **练习页组件替换**
   - 理由: 练习页是核心功能页面
   - 数据支撑: 竞品使用卡片式布局提升答题体验
   - 技术方案: 使用 BubbleCard, BubbleField

## 技术实现参考

### v0 组件导入方式
```javascript
// 在页面中导入 v0 组件
import WelcomeBanner from '@/components/v0/WelcomeBanner.vue'
import StatsCard from '@/components/v0/StatsCard.vue'
import KnowledgeBubble from '@/components/v0/KnowledgeBubble.vue'
import BubbleCard from '@/components/v0/BubbleCard.vue'
import BubbleField from '@/components/v0/BubbleField.vue'
import ActivityItem from '@/components/v0/ActivityItem.vue'

export default {
  components: {
    WelcomeBanner,
    StatsCard,
    // ...
  }
}
```

### 关键技术点
1. **组件注册**: 在 components 选项中注册
2. **Props 传递**: 根据组件 API 传递数据
3. **样式覆盖**: 使用 :deep() 选择器覆盖组件样式
4. **主题适配**: 使用 CSS 变量支持深色模式
