- ✅ **显示预计剩余时间** - 通过

  - importDetailText 显示："预计剩余 X 秒"
  
- ⚠️ **显示已上传大小/总大小** - 部分通过

- ✅ **提供取消上传的按钮** - 通过 ⭐
  - AI 加载遮罩中添加"取消"按钮
  - 点击后弹出确认对话框
  - 确认后停止生成，保留已生成的题目

### US-003: 失败重试机制
- ✅ **上传失败后显示重试按钮** - 通过 ⭐
  - errorInfo.canRetry 为 true 时显示错误卡片
  - 错误卡片包含"重试"按钮
  - 点击重试后自动重新开始生成
  
- ✅ **保留已选择的文件信息** - 通过
  - fileName 和 fullFileContent 在失败后保留
  - 重试时无需重新选择文件
  
- ⚠️ **重试时从断点继续** - 部分支持
  - 网络超时会自动静默重试
  - 其他错误需要手动重试，从头开始
