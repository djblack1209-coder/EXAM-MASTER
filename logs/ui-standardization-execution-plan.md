## 🔧 标准化规则

### 颜色替换映射表

#### 主题色
- `#00a96d` → `var(--brand-color)`
- `#42E695` → `var(--brand-light)`
- `#3BB2B8` → `var(--brand-accent)`

#### 文本颜色
- `#333` / `#212529` → `var(--text-primary)`
- `#666` / `#495057` → `var(--text-secondary)`
- `#999` / `#9e9e9e` → `var(--text-tertiary)`
- `#fff` / `#ffffff` → `var(--text-inverse)`

#### 背景色
- `#fff` / `#ffffff` → `var(--bg-primary)`
- `#f5f5f5` / `#f8f9fa` → `var(--bg-secondary)`
- `#e9ecef` → `var(--bg-tertiary)`

#### 边框色
- `#e9ecef` / `#dee2e6` → `var(--card-border)`
- `#ddd` → `var(--border-light)`

#### 阴影
- `rgba(0, 0, 0, 0.08)` → `var(--shadow-sm)`
- `rgba(0, 0, 0, 0.1)` → `var(--shadow-md)`
- `rgba(0, 0, 0, 0.15)` → `var(--shadow-lg)`

#### 状态色
- 成功: `#28a745` → `var(--success-green)`
- 警告: `#ffc107` → `var(--warning-yellow)`
- 危险: `#ff453a` → `var(--danger-red)`
- 信息: `#17a2b8` → `var(--info-blue)`

#### 特殊效果（可保留）
- 粒子效果颜色
- Canvas绘制颜色
- 特定动画渐变色（需评估）

3. **替换颜色** - 使用CSS变量替换
4. **验证构建** - 确保无语法错误
5. **手动测试** - 验证视觉效果
6. **更新进度** - 标记完成状态
