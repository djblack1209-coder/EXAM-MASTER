# TabBar 图标说明

此目录用于存放底部导航栏图标。

## 所需图标清单

请准备以下图标文件（建议尺寸：81x81px，PNG格式）：

### 1. 首页图标
- `home.png` - 未选中状态（灰色 #999999）
- `home-active.png` - 选中状态（绿色 #07C160）

### 2. 刷题图标
- `practice.png` - 未选中状态（灰色 #999999）
- `practice-active.png` - 选中状态（绿色 #07C160）

### 3. 择校图标
- `school.png` - 未选中状态（灰色 #999999）
- `school-active.png` - 选中状态（绿色 #07C160）

### 4. 设置图标
- `profile.png` - 未选中状态（灰色 #999999）
- `profile-active.png` - 选中状态（绿色 #07C160）

## 临时方案

如果暂时没有图标，可以使用以下两种方案：

1. **使用纯色方块作为占位符**（已在下方创建）
2. **使用 Iconfont 字体图标**（需要额外配置）

## 注意事项

- 图标建议使用 PNG 格式，支持透明背景
- 建议尺寸：81x81px @ 1x，162x162px @ 2x，243x243px @ 3x
- 保持图标风格统一，线条粗细一致
