# 静态资源说明

## 图标文件需求

请在对应目录下放置以下图标文件：

### /static/icons/ 目录
1. **gift.png** - 礼物图标（用于导航栏邀请好友按钮）
   - 尺寸：建议 64x64px
   - 颜色：白色或透明背景
   - 用途：HomeNavbar 组件

2. **arrow-right.png** - 右箭头图标（用于刷题入口）
   - 尺寸：建议 64x64px
   - 颜色：白色
   - 用途：PracticeBanner 组件

3. **check.png** - 勾选图标（用于待办列表）
   - 尺寸：建议 48x48px
   - 颜色：白色
   - 用途：TodoList 组件完成状态

### /static/ 根目录
1. **logo.png** - 应用主Logo
   - 尺寸：建议 256x256px
   - 样式：Exam-Master 品牌Logo（绿色书本向上箭头）
   - 用途：首页Logo展示

## 临时方案

如果暂时没有图标文件，可以：
1. 使用纯CSS绘制简单图标
2. 使用 uni-icons 字体图标库
3. 从 iconfont 或其他图标库下载

## 颜色规范
- 主色调：#07C160（微信绿）
- 辅助色：#05A850（深绿）
- 文字色：#333333（主文字）、#999999（次要文字）
- 背景色：#F8F9FA（浅灰背景）
