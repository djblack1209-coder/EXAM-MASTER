# 修复和设计计划

## 1. 修复首页学习时长的UI显示问题

**问题分析**：从截图可以看出，首页学习时长的数字和图表重叠，导致显示混乱。

**解决方案**：
- 调整统计卡片的布局结构，将图表和数字显示分离
- 优化图表大小和位置，确保不与数字重叠
- 改进学习时长的样式，使其更加清晰易读
- 统一使用CSS变量控制样式，确保一致性

## 2. 重新设计"刷题页面"

**设计参考**：wise.com的简洁、现代化设计风格

**功能保留**：
- 题库状态显示
- 开始刷题按钮
- 导入资料选项
- 错题本入口
- 学习进度显示

**设计改进**：
- 使用wise.com的配色方案（主色调：绿色#00a96d，辅助色：蓝色#005bea）
- 采用卡片式布局，增强层次感
- 添加平滑的动画效果
- 优化按钮和交互元素的设计
- 改进加载状态的显示

## 3. 重新设计"设置页面"

**设计参考**：wise.com的简洁、现代化设计风格

**设计改进**：

## 4. 扫描项目功能，确保UI完整性

**检查内容**：
- 检查所有页面和组件的功能

**预期结果**：
- 首页学习时长显示清晰，图表和数字不重叠

- 整体设计风格统一，交互体验流畅
