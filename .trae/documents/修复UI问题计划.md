# 修复UI问题计划

## 1. 深色模式下首页是白色没有跟随变更
- **问题分析**：首页container元素已有dark-mode类绑定，但深色模式样式可能未正确应用
- **修复方案**：检查并调整首页CSS中深色模式的样式定义，确保CSS变量和选择器正确
- **涉及文件**：`/pages/index/index.vue`

## 2. 左上角用户名及头像不清晰
- **问题分析**：头像可能存在模糊效果或样式问题
- **修复方案**：调整头像样式，移除不必要的模糊效果，优化图片显示质量
- **涉及文件**：`/pages/index/index.vue`

## 3. 右上角刷新按钮被微信原生按钮遮挡
- **问题分析**：刷新按钮位置与微信胶囊按钮重叠
- **修复方案**：调整刷新按钮的右外边距，确保与微信胶囊按钮保持安全距离
- **涉及文件**：`/pages/index/index.vue`

## 4. 点击“每日金句”不能切换
- **问题分析**：每日金句组件缺少点击事件处理
- **修复方案**：为每日金句卡片添加点击事件，实现金句切换功能
- **涉及文件**：`/pages/index/index.vue`

## 5. 深色模式下“择校”页面是白色
- **问题分析**：择校页面dark-mode类已绑定，但深色模式样式可能未正确定义
- **修复方案**：检查并调整择校页面CSS中深色模式的样式定义
- **涉及文件**：`/pages/school/index.vue`

## 6. 深色模式下“刷题中心”是白色
- **问题分析**：刷题中心页面可能缺少深色模式支持
- **修复方案**：为practice-container添加深色模式样式，确保isDark状态正确设置
- **涉及文件**：`/pages/practice/index.vue`

## 7. 非深色模式下，各个页面的悬浮导航栏都是黑色
- **问题分析**：导航栏缺少浅色模式下的样式定义
- **修复方案**：为导航栏添加浅色模式样式，区分深色和浅色模式下的导航栏外观
- **涉及文件**：所有包含导航栏的页面（首页、择校、刷题中心等）

## 实施步骤
1. 修复首页的深色模式问题和刷新按钮位置
2. 修复用户名及头像清晰度问题
3. 为每日金句添加切换功能
4. 修复择校页面的深色模式问题
5. 修复刷题中心的深色模式问题
6. 修复所有页面导航栏的浅色模式样式
7. 测试所有修复效果，确保UI在各种模式下正常显示

## 预期效果
- 深色模式下所有页面背景色正确切换
- 用户名及头像清晰显示
- 刷新按钮不被微信原生按钮遮挡
- 每日金句可点击切换
- 导航栏在不同模式下显示正确颜色
