## 三、四端打包方案

### 现状分析

- **微信小程序**: 已有构建流程，但主包超限（需优化）
- **H5/PWA**: 已配置 VitePWA，但未实际部署
- **Android/iOS App**: uni-app 原生支持，但未配置
- **Mac/Win 桌面端**: 无任何支持

### 技术方案

| 平台        | 方案               | 工具链                      | 打包产物        |
| ----------- | ------------------ | --------------------------- | --------------- |
| **Android** | uni-app → App 构建 | HBuilderX CLI / uni-app CLI | `.apk` / `.aab` |
| **iOS**     | uni-app → App 构建 | HBuilderX CLI + Xcode       | `.ipa`          |
| **Windows** | H5 + Electron 封装 | electron-builder            | `.exe` 安装包   |
| **macOS**   | H5 + Electron 封装 | electron-builder            | `.dmg` 安装包   |

### Electron 桌面端方案（搬运优化）

- 基于 `electron-vite` 开源项目（GitHub 3k+ stars）
- 直接包装 H5 构建产物，零额外开发
- 自动更新：集成 `electron-updater`
- 跨平台构建：GitHub Actions 一键出包
