## 三、运行步骤

### 1. 安装依赖
```bash
# 如果还没有安装依赖
npm install
```

### 2. 准备图标资源
在 `/static/icons/` 目录下添加以下图标：
- `gift.png` - 礼物图标（白色，64x64px）
- `arrow-right.png` - 右箭头（白色，64x64px）
- `check.png` - 勾选图标（白色，48x48px）

在 `/static/` 目录下添加：
- `logo.png` - 应用Logo（256x256px）

> 💡 **临时方案**：如果暂时没有图标，页面仍可正常运行，只是图标位置会显示占位符。

```bash

```

1. 打开微信开发者工具
2. 导入项目：选择 `dist/dev/mp-weixin` 目录
3. 查看首页效果

## 四、核心技术点

### 1️⃣ 导航栏胶囊适配（重点）

**问题**：微信小程序右上角有固定的胶囊按钮，自定义导航栏需要避开它。

**解决方案**：
```javascript
// 使用 useNavbar hook
const { navbarHeight, menuButton } = useNavbar()

// 右侧按钮定位计算
const rightButtonStyle = computed(() => {
  return {
    top: gap + 'px',
    right: (menuButton.value.width + 15) + 'px'  // 胶囊宽度 + 安全间距
  }
})
```

**关键数值**：
- 胶囊宽度：约87px（不同设备可能不同）
- 安全间距：15px（确保不被遮挡）
- 总偏移：right = 87 + 15 = 102px

---

### 2️⃣ 组件化开发

所有UI模块都拆分为独立组件，便于维护和复用：

```vue
<template>
  <view class="home-page">
    <!-- 导航栏 -->
    <HomeNavbar @invite-friend="handleInviteFriend" />
    
    <!-- 倒计时 -->
    <CountdownCard :target-date="examDate" />
    
    <!-- 刷题入口 -->
    <PracticeBanner @banner-tap="handlePracticeTap" />
    
    <!-- 待办列表 -->
    <TodoList :todos="todoList" @toggle-todo="handleToggleTodo" />
  </view>
</template>
```

---

### 3️⃣ 数据绑定与交互

```javascript
// 待办列表切换
const handleToggleTodo = (id) => {
  const todo = todoList.value.find(item => item.id === id)
  if (todo) {
    todo.completed = !todo.completed
    
    // 震动反馈
    uni.vibrateShort({ type: 'light' })
  }
}
```

## 七、常见问题

### Q1: 导航栏右侧按钮被胶囊遮挡？
**A:** 检查 `HomeNavbar.vue` 中的 `rightSafeMargin` 值，可以适当增大：
```javascript
const rightSafeMargin = 20  // 增大到20px
```

### Q2: 页面顶部被遮挡？
**A:** 确保在 `pages.json` 中设置了自定义导航栏：
```json
{
  "path": "pages/index/index",
  "style": {
    "navigationStyle": "custom"
  }
}
```

### Q3: 图标不显示？
**A:** 
1. 检查图标文件路径是否正确
2. 确保图标文件已放入 `/static/icons/` 目录
3. 临时可以使用 uni-icons 替代

### Q4: 倒计时天数不对？
**A:** 检查系统时间，计算基于当前日期和目标日期的差值。
