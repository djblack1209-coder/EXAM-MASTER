# Raycast 日志复制使用指南

## 📋 简介

Raycast 是一个强大的 macOS 生产力工具，可以方便地复制 Console 日志。

## 🚀 快速开始

### 步骤 1: 安装 Raycast

1. 访问 [Raycast 官网](https://www.raycast.com/)
2. 下载并安装 Raycast
3. 完成初始设置

### 步骤 2: 使用 Raycast 复制日志

#### 方法 1: 使用剪贴板历史（推荐）

1. **在微信开发者工具的 Console 中**：
   - 使用鼠标选择要复制的日志内容
   - 按 `Cmd+C` 复制

2. **在 Raycast 中**：
   - 按 `Cmd+Space` 打开 Raycast
   - 输入 "Clipboard History" 或 "剪贴板历史"
   - 选择最近复制的日志内容
   - 按回车粘贴

#### 方法 2: 使用 Raycast 的文本选择功能

1. **在微信开发者工具的 Console 中**：
   - 使用鼠标选择日志内容
   - 按 `Cmd+C` 复制

2. **在 Raycast 中**：
   - 按 `Cmd+Space` 打开 Raycast
   - 输入 "Paste" 或直接粘贴（`Cmd+V`）

### 步骤 3: 批量复制（如果需要）

如果日志很多，可以：

1. **分段复制**：
   - 在 Console 中，先选择一部分日志复制
   - 再选择下一部分复制
   - Raycast 会保存所有剪贴板历史

2. **使用 Raycast 的文本合并功能**：
   - 在 Raycast 中搜索 "Text" 相关功能
   - 可以合并多个剪贴板内容

## 💡 使用技巧

### 技巧 1: 快速访问剪贴板历史

- **快捷键**：`Cmd+Shift+V`（如果已配置）
- **或**：在 Raycast 中搜索 "Clipboard"

### 技巧 2: 搜索剪贴板历史

- 在 Raycast 的剪贴板历史中，可以直接搜索关键词
- 例如：搜索 "school" 可以快速找到择校相关的日志

### 技巧 3: 固定常用日志

- Raycast 支持固定剪贴板条目
- 可以固定常用的日志模板或格式

## 🔧 高级功能

### 自定义快捷键

1. 打开 Raycast 设置
2. 找到 "Clipboard History" 扩展
3. 设置自定义快捷键（如 `Cmd+Shift+C`）

### 使用 Raycast 扩展

Raycast 支持丰富的扩展，可以：
- 格式化 JSON
- 转换文本格式
- 合并多个剪贴板内容

## 📝 注意事项

1. **隐私**：Raycast 的剪贴板历史可能包含敏感信息，请注意隐私设置
2. **存储**：Raycast 会保存剪贴板历史，注意定期清理
3. **性能**：如果日志非常大，建议分段复制
