## 样式规范总结

| 类型 | 变量/值 | 说明 |
|------|---------|------|
| 主题色 | `#07C160` | 绿色，用于按钮和选中状态 |
| 背景色 | `#F5F7FA` | 浅灰，页面背景 |
| 卡片背景 | `#FFFFFF` | 白色 |
| 卡片圆角 | `24rpx` | 圆角大小 |
| 卡片阴影 | `0 4rpx 20rpx rgba(0,0,0,0.04)` | 柔和阴影 |
| 组件间距 | `30rpx` | 垂直间距 |
| 文字主色 | `#333333` | 深灰 |
| 文字副色 | `#666666` | 中灰 |
| 文字辅色 | `#999999` | 浅灰 |

## 技术栈

- **框架**：uni-app（Vue 3 + Composition API）
- **状态管理**：Vuex 4
- **样式**：SCSS
- **UI 风格**：现代扁平化设计
- **图标方案**：Emoji + PNG（TabBar）

## 性能优化建议

1. **组件按需加载**：已使用局部注册，避免全局注册
2. **图标优化**：使用 Emoji 替代图片，减少资源加载
3. **CSS 渐变**：使用 CSS 渐变替代渐变图片
4. **骨架屏**：建议为刷题页添加加载骨架屏
5. **懒加载**：最近练习列表数据建议使用懒加载
