# 第8阶段功能使用指南

## 一、开屏页功能

### 功能说明
应用启动时会自动显示开屏页,展示品牌Logo和加载动画。

### 视觉效果
1. **背景**: 绿色全屏背景 (#07C160)
2. **Logo**: 白色线框书本图标 + "Exam-Master"文字
3. **加载动画**: 三个白色圆点跳动
4. **底部装饰**: 彩色流体波浪 + 动态箭头
5. **跳过按钮**: 右上角半透明按钮

### 交互逻辑
- **自动跳转**: 3秒后自动进入首页
- **手动跳过**: 点击右上角"跳过"按钮立即进入

### 代码位置
- 文件: `pages/splash/index.vue`
- 路由: 已设置为启动首页(pages.json第一位)

## 二、邀请弹窗功能

### 功能说明
在设置页面邀请好友使用应用,通过分享链接或邀请码。

### 打开方式
1. 进入"设置"Tab
2. 找到"我的好友们"区域
3. 点击右上角绿色"邀请好友"按钮

### 弹窗内容
1. **邀请码**: `EXAM2026` (绿色虚线框)
2. **邀请链接**: `https://exam-master.com/i/EXAM...`
3. **操作按钮**:
   - 复制链接按钮
   - 分享给微信好友 (预埋)
   - 生成分享海报 (预埋)

### 功能状态
- ✅ **已实现**: 复制链接功能
- ⏳ **开发中**: 微信分享功能
- ⏳ **开发中**: 海报生成功能

### 关闭方式
- 点击右上角 ✕ 按钮
- 点击弹窗外部遮罩层
- 完成操作后自动关闭

## 四、自定义配置

### 修改开屏页时长
编辑 `pages/splash/index.vue`:

```javascript
onMounted(() => {
  timer = setTimeout(() => {
    handleSkip()
  }, 5000)  // 改为5秒

```

### 修改邀请码
编辑 `components/InviteModal.vue`:

```vue
<!-- 修改邀请码显示 -->
<text class="invite-code">YOUR_CODE</text>

<!-- 修改复制的链接 -->
<script>
const handleCopy = () => {
  uni.setClipboardData({
    data: 'https://exam-master.com/i/YOUR_CODE'
  })

```

### 修改主题色
所有绿色使用的色值:
- 主色: `#07C160`
- 按钮色: `#52C41A`
- 按钮激活色: `#06AD51`

可全局替换这些颜色值来改变主题。

## 五、常见问题

### Q1: 开屏页不显示?
**A**: 检查 `pages.json` 确保 `pages/splash/index` 在数组第一位。

### Q2: 邀请弹窗点击按钮无反应?
**A**: 检查父组件是否正确传递了 `v-model:visible` 属性。

### Q3: 如何禁用开屏页?
**A**: 在 `pages.json` 中将 `pages/index/index` 移到第一位。

### Q4: 微信分享功能何时可用?
**A**: 需要在微信小程序环境下配置分享参数,目前为预埋代码。

### Q5: 如何修改动画速度?
**A**: 编辑对应文件中的 `animation` 属性,修改 `duration` 值。
