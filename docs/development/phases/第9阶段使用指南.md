## 🐛 常见问题排查

### Q1: 海报显示空白
**可能原因**:
- Canvas 未正确初始化
- 绘图逻辑未执行

**解决方案**:
- 检查控制台是否有错误
- 确认 `drawPoster()` 方法被调用
- 检查 Canvas ID 是否正确：`sharePoster`

### Q2: 保存到相册失败
**可能原因**:
- 没有相册权限
- Canvas 未绘制完成就保存

**解决方案**:
- 检查 manifest.json 中的权限配置
- 增加延迟时间：`setTimeout(..., 500)`
- 在真机上测试（小程序开发工具可能不支持）

### Q3: 提交后没有跳转
**可能原因**:
- 路由配置问题
- TabBar 配置问题

**解决方案**:
- 使用 `uni.switchTab` 而不是 `uni.navigateTo`
- 确认 pages.json 中首页配置正确

### Q4: 上一步按钮不工作
**可能原因**:
- 事件没有正确绑定
- currentStep 没有正确更新

**解决方案**:
- 检查 `@prev` 事件是否绑定
- 检查 `handlePrev` 方法实现
- 查看控制台 currentStep 值
