## 一、开屏页实现 (pages/splash/index.vue)

### 1. UI设计实现

#### 整体布局
- **背景色**: #07C160 (品牌绿色)
- **全屏展示**: 100vw × 100vh
- **垂直居中**: Flexbox布局

#### Logo区域
- **书本图标**: 
  - 使用CSS绘制白色线框风格的书本
  - 左右两页使用边框绘制
  - 顶部添加向上箭头,象征进步
- **应用名称**: "Exam-Master"
  - 字号: 68rpx
  - 颜色: 白色
  - 字间距: 4rpx

#### 加载动画
- **三个圆点**: 白色,跳动动画
- **动画效果**: 
  - 使用CSS keyframes实现弹跳效果
  - 三个点依次延迟触发
  - 循环播放,视觉反馈流畅

#### 底部装饰
- **波浪纹理**: 
  - 渐变色彩流体效果
  - 从蓝色到黄色到橙色的过渡
  - 椭圆径向渐变模拟水波
  - 脉冲动画增强视觉效果
- **大箭头**: 
  - 半透明白色
  - 指向右侧
  - 平滑左右移动动画

#### 跳过按钮
- **位置**: 右上角
- **样式**: 半透明白色背景 + 毛玻璃效果
- **交互**: 点击立即跳转

### 2. 逻辑实现

```javascript
// 自动跳转逻辑
onMounted(() => {
  timer = setTimeout(() => {
    uni.switchTab({ url: '/pages/index/index' })
  }, 3000)
})

// 手动跳过
const handleSkip = () => {
  clearTimeout(timer)
  uni.switchTab({ url: '/pages/index/index' })
}
```

```json

  "style": {
    "navigationBarTitleText": "Exam-Master",
    "navigationStyle": "custom",
    "navigationBarBackgroundColor": "#07C160"
```

## 三、集成到设置页

### 1. 引入组件
```vue
<script setup>
import { ref } from 'vue'
import InviteModal from '@/components/InviteModal.vue'

const showInviteModal = ref(false)
</script>
```

### 2. 使用组件
```vue
<template>
  <!-- 好友列表组件发射invite事件 -->
  <FriendsList @invite="showInviteModal = true" />
  
  <!-- 邀请弹窗 -->
  <InviteModal 
    v-model:visible="showInviteModal"
    @close="showInviteModal = false"
  />
</template>
```

### 3. FriendsList组件改造
```vue
<script setup>
import { defineEmits } from 'vue'

const emit = defineEmits(['invite'])

const handleInvite = () => {
  emit('invite')  // 触发父组件显示弹窗
}
</script>
```

## 四、技术亮点

### 1. CSS动画
- **跳动圆点**: keyframes实现循环弹跳
- **波浪脉冲**: 缩放+位移+透明度组合动画
- **箭头移动**: 平移动画增强引导感
- **所有动画**: 使用ease-in-out缓动,视觉流畅

### 2. 响应式设计
- 使用rpx单位适配不同屏幕
- Flexbox布局确保居中对齐
- 点击态反馈 (:active伪类)

### 3. 组件通信
- **Props/Emits**: 标准Vue3组件通信
- **v-model:visible**: 双向绑定控制显示状态
- **事件冒泡**: FriendsList → Profile → InviteModal

### 4. 用户体验
- **自动跳转**: 3秒后自动进入主页
- **手动跳过**: 右上角跳过按钮
- **点击遮罩关闭**: 弹窗外部点击关闭
- **阻止冒泡**: 弹窗内部点击不关闭 (@click.stop)
- **复制反馈**: Toast提示操作结果

## 六、使用说明

### 启动流程
1. 应用启动 → 显示开屏页 (3秒)
2. 自动跳转或手动跳过 → 进入首页

### 邀请流程
1. 进入设置页
2. 找到"我的好友们"区域
3. 点击右上角"邀请好友"按钮
4. 弹出邀请弹窗
5. 可选操作:
   - 复制链接
   - 分享给微信好友
   - 生成分享海报
