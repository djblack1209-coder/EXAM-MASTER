# 第7阶段开发总结：AI聊天主界面与API对接

## 📋 开发概述

本阶段完成了AI聊天功能的核心界面开发,包括仿微信风格的聊天页面、与智谱AI的API对接,以及从刷题页到聊天页的导航流程。

## ✅ 完成的功能

### 1. 聊天页面 (`pages/chat/chat.vue`)

#### 页面结构
- **自定义导航栏**
  - 显示"AI助教"标题和"在线"状态
  - 左侧返回按钮
  - 适配刘海屏/安全区域

- **消息列表区域**
  - 使用 `scroll-view` 实现可滚动的消息列表
  - 自动滚动到底部 (使用 `scroll-into-view`)
  - 初始欢迎消息："同学你好,我是你的考研AI助教,有什么不懂的都可以问我!"

- **消息气泡样式**
  - **用户消息**: 绿色背景 (#95EC69),靠右对齐,带右侧小三角
  - **AI消息**: 白色背景,靠左对齐,带左侧小三角
  - **错误消息**: 红色文字显示错误提示
  - **加载动画**: AI思考时显示三个跳动的圆点

- **底部输入栏**
  - 固定在底部,适配安全区域
  - 输入框 + 发送按钮
  - 发送按钮根据输入状态和加载状态自动禁用/启用

#### 核心业务逻辑

```javascript
// 状态管理
- messages: 对话历史数组
- inputText: 输入框绑定值
- isLoading: 加载状态控制
- scrollToView: 滚动位置控制

// 发送消息流程
1. 验证输入是否为空
2. 添加用户消息到 messages
3. 清空输入框并滚动到底部
4. 设置 isLoading = true
5. 调用 ZhipuService.sendMessage(messages)
6. 解析响应并添加AI回复
7. 错误处理：显示友好的错误提示
8. finally: 恢复状态并滚动到底部
```

## 🔧 技术实现

### 1. API对接

```javascript
import ZhipuService from '@/services/zhipuService.js'

// 发送消息
const response = await ZhipuService.sendMessage(messages.value)

// 响应处理
if (response.choices && response.choices[0]) {
  const aiMessage = response.choices[0].message.content
  messages.value.push({
    role: 'assistant',
    content: aiMessage
  })
}
```

### 2. 自动滚动实现

```javascript
const scrollToBottom = () => {
  nextTick(() => {
    const lastIndex = messages.value.length - 1
    scrollToView.value = 'msg-' + lastIndex
  })
}
```

### 3. 计算属性控制发送按钮

```javascript
const canSend = computed(() => {
  return inputText.value.trim().length > 0 && !isLoading.value
})
```
