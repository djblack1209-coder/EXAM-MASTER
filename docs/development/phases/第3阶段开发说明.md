## 技术要点

### 1. 组件通信
- 当前组件为独立展示组件，暂无父子通信需求
- 后续可通过 `props` 传递数据，`emit` 触发父组件事件

### 2. 状态管理
- 错题数据建议存储在 `stores/modules/study.js`
- AI 解锁状态建议存储在 `stores/modules/user.js`

### 3. 文件上传
- 使用 `uni.chooseFile()` 或 `uni.chooseMessageFile()` 选择文件
- 支持格式：`.txt`, `.docx`, `.pdf`
- 需要后端 API 支持文件解析

### 4. 样式优化
- 已使用 CSS 渐变替代图片，减少资源加载
- 已添加交互动效（按钮点击缩放、输入框焦点高亮）
- 已适配底部安全区域（iPhone X 及以上机型）

## 注意事项

⚠️ **图标问题**：
- TabBar 需要本地图标文件才能正常显示
- 当前配置的图标路径为 `static/tabbar/*.png`
- 如图标文件不存在，TabBar 文字仍可正常显示，但图标位置会空白
- 请参照 `static/tabbar/README.md` 准备图标文件

⚠️ **首页导航栏**：
- 首页保留了 `navigationStyle: "custom"` 自定义导航栏
- 其他三个页面使用原生导航栏
- 如需统一风格，可在后续阶段调整

⚠️ **数据模拟**：
- 当前所有数据为硬编码（如错题数 47、电池百分比 75%）
- 后续需要对接 Vuex 状态管理和后端 API
