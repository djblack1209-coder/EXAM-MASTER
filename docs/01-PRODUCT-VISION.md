# 产品愿景与战略方向

> 更新日期：2026-04-25

## 一、产品定位

**考研大师** — 基于百度网盘资源库的考研备考闪卡刷题工具。

核心理念：不做内容生产者，做内容加工工具。将百度网盘中的机构PDF资料离线加工为结构化闪卡题目，供用户在碎片时间刷题复习。

## 二、核心痛点

1. 考研资料过多，散落在PDF和纸质材料中，无法在碎片时间使用
2. 市场上没有覆盖考研全科目的、好看好用的、带间隔重复(FSRS)的刷题工具
3. 现有产品要么只覆盖公共课(粉笔)，要么体验差(Anki)，要么只有英语(百词斩)

## 三、技术架构

### 数据流

```
闲鱼资源库 → baidu-autosave定时自动转存 → 你的百度网盘SVIP
                                              ↓
                                 AI离线加工(PDF→结构化JSON闪卡)
                                              ↓
                                 资源索引上传到Laf云数据库(几百KB)
                                              ↓
                                 用户APP加载索引→刷闪卡→记录进度
```

### 关键约束

- **版权保护**：用户永远看不到原始PDF，只看到加工后的闪卡
- **隐私隔离**：只按学科分类，不暴露机构/老师信息
- **轻量服务器**：Laf只存索引和用户进度数据（每用户几十KB）
- **免费存储**：百度网盘SVIP作为CDN和原始资料存储

### 百度网盘集成方案

- 百度网盘开放API：读取文件列表、获取下载链接
- 目录限制：第三方应用只能访问 `/apps/考研大师/` 目录（天然隐私隔离）
- `baidu-autosave` (GitHub, MIT): 分享链接定时自动转存，智能去重
- `BaiduPCS-Py` (GitHub, MIT): 百度网盘Python API，命令行操作

### 资料清洗流程

1. `baidu-autosave` 转存时正则过滤（只要.pdf/.doc，忽略视频）
2. 清洗脚本：删除广告文件（文件名含"报名/咨询/加群"等关键词）
3. 删除<50KB的PDF（大概率是广告单页）
4. 按学科/年份自动整理目录结构
5. 生成资源索引JSON上传Laf

## 四、平台策略

| 平台 | 定位 | 功能范围 | 页面数 |
|------|------|---------|--------|
| iOS / Android | 主力产品 | 全功能、最精美UI、完整游戏化、离线缓存 | 5-6个 |
| 微信小程序 | 轻量入口 | 极简版：每日快速刷题 + 引导下载APP | 3个 |

微信小程序目标包体积：< 500KB。

## 五、刷题题型设计

| 题型 | 交互方式 | 适用场景 |
|------|---------|---------|
| 经典闪卡 | 翻转 + 自评(会/模糊/不会) | 概念记忆 |
| 四选一 | 点选 | 标准选择题 |
| 连线配对 | 拖拽连线 | 概念匹配 |
| 排序题 | 拖拽排序 | 时间线/步骤 |
| 判断题 | 左滑错/右滑对 | 对错判断 |
| 填空选词 | 从选项中选词填入 | 关键词记忆 |

所有交互都是点、滑、拖——不需要手动输入，适合手机碎片时间。

## 六、核心技术壁垒

- **FSRS间隔重复算法**：已实现(ts-fsrs)，大多数竞品没有
- **百度网盘集成**：免费获得近乎无限的存储和CDN能力
- **AI加工管线**：将PDF批量转为结构化闪卡的自动化流程
- **游戏化刷题体验**：多邻国级别的交互设计

## 七、商业模式

| 层级 | 价格 | 内容 |
|------|------|------|
| 免费 | 0 | 每科前200张闪卡 + 每日限刷50张 |
| 会员 | 19.9/月 或 128/年 | 全部题库 + 无限刷题 + 高级题型 + 离线缓存 |
| 单科购买 | 6.9/科 | 永久解锁某一科全部闪卡 |

## 八、运营成本

| 项目 | 成本 |
|------|------|
| 百度网盘SVIP | 已有 |
| 闲鱼资源采购 | 首次几十元，后续偶尔补购 |
| Laf云数据库 | 免费额度内 |
| AI加工(GPT-4o-mini) | 每科约几元 |
| 服务器 | 接近零 |

## 九、竞品差异

| 维度 | 粉笔 | Anki | 百词斩 | 考研大师 |
|------|------|------|--------|---------|
| 科目覆盖 | 公共课 | 用户自建 | 仅英语 | 全科目(机构资料加工) |
| 预制内容 | 有 | 无 | 有 | 有 |
| FSRS算法 | 无 | 有 | 无 | 有 |
| 多题型 | 选择题为主 | 翻卡 | 游戏化 | 6种题型 |
| UI体验 | 中等 | 差 | 优秀 | 目标:优秀 |
| 价格 | 高(几百-几千) | 免费 | 中 | 低(6.9-19.9) |

## 十、冷启动策略

1. 小红书/B站发帖："把考研资料变成手机能刷的闪卡"
2. 考研论坛种子用户招募（前100名永久免费）
3. 产品内裂变：分享给研友，双方各得额外刷题次数
4. 自己作为考研过来人产出真实使用体验
