## 🐛 遇到的问题

### 编译错误
```
[ WXSS 文件编译错误] 
./pages/splash/index.wxss(453:1): unexpected token `*`
(env: macOS,mp,2.01.2510260; lib: 3.13.0)
```

### 错误原因
微信小程序的 WXSS 编译器不支持某些现代 CSS 特性：
1. ❌ 媒体查询中的通配符选择器 `*`
2. ❌ `prefers-reduced-motion` 媒体查询
3. ❌ `prefers-contrast` 媒体查询
4. ❌ `prefers-color-scheme` 媒体查询（部分版本）
5. ⚠️ `clamp()` CSS 函数（部分版本不支持）
6. ❌ 非 WebKit 浏览器前缀（`-moz-`, `-ms-`）
7. ❌ `cursor` 属性（小程序无意义）
8. ⚠️ `touch-action` 属性（部分版本不支持）

## ✅ 修复方案

### 1. 移除不支持的媒体查询

#### ❌ 修复前
```scss
/* 尊重用户的动画偏好设置 */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
  }
  
  .logo-container {
    animation: none !important;
  }
}

/* 高对比度模式支持 */
@media (prefers-contrast: high) {
  .app-name {
    text-shadow: 0 2rpx 8rpx rgba(0, 0, 0, 0.8);
  }
}

/* 深色模式支持 */
@media (prefers-color-scheme: dark) {
  .splash-page {
    background: linear-gradient(...);
  }
}
```

#### ✅ 修复后
```scss
/* 
 * 注意：微信小程序 WXSS 不支持以下媒体查询：
 * - prefers-reduced-motion（减少动画偏好）
 * - prefers-contrast（对比度偏好）
 * - prefers-color-scheme（深色模式）在部分版本支持有限
 * 
 * 替代方案：
 * 1. 在 JavaScript 中通过 wx.getSystemInfo() 获取系统主题
 * 2. 动态添加类名来切换样式
 * 3. 提供应用内设置选项让用户手动切换
 */
```

## 🔧 未来的改进方案

### 如果需要深色模式支持

```javascript
// 在 pages/splash/index.vue 的 <script setup> 中添加
import { ref, onMounted } from 'vue'

const isDarkMode = ref(false)

onMounted(() => {
  // 获取系统主题
  wx.getSystemInfo({
    success: (res) => {
      isDarkMode.value = res.theme === 'dark'
    }
  })
  
  // 监听主题变化
  wx.onThemeChange((res) => {
    isDarkMode.value = res.theme === 'dark'
  })
})
```

```vue
<template>
  <view class="splash-page" :class="{ 'dark-mode': isDarkMode }">
    <!-- 内容 -->
  </view>
</template>

<style lang="scss" scoped>
.splash-page {
  background: linear-gradient(
    to bottom,
    #00E676 0%,
    #00C853 100%
  );
  
  &.dark-mode {
    background: linear-gradient(
      to bottom,
      #00C853 0%,
      #009E3E 100%
    );
  }
}
</style>
```
