# 微信小程序UI重设计规格书

> 状态：待执行
> 优先级：最高
> 目标：3页面极简小程序，包体积<500KB

## 一、设计原则

- **移动端优先**：375px宽度，全部交互适配触摸操作
- **小程序规范**：遵循微信小程序设计规范，使用原生导航栏和TabBar
- **绿色主题**：主色 #00B86B（Wise风格），白色底色，蓝色辅助
- **极简**：每个页面只做一件事，没有视觉噪音
- **无美术资源依赖**：纯CSS实现所有视觉效果，不依赖图片（避免留白）

## 四、组件要求

### 不使用的
- ❌ 不用任何图片/SVG图标（用emoji或纯CSS图标）
- ❌ 不用第三方UI库的复杂组件
- ❌ 不用动画库
- ❌ 不用ECharts或图表库

### 使用的
- ✅ 微信原生导航栏（page.json中配置）
- ✅ 微信原生TabBar
- ✅ 纯CSS进度条、卡片、按钮
- ✅ wot-design-uni 只用基础组件（按钮、图标、弹窗）
- ✅ CSS transition 做简单过渡动画

## 六、执行要点

1. **先改 pages.json**：只保留3个Tab页+必要的子页面路由
2. **从首页开始**：逐页重写template和style，保留script逻辑
3. **不引入新依赖**：纯CSS + uni-app原生API
4. **每页完成后验证**：npm run build:mp-weixin 检查包体积
5. **目标包体积**：主包<500KB
