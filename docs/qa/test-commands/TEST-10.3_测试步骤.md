## 🔍 问题排查

### 问题 1：没有看到 `[TEST-10.3] 📊 获取到当前总分数` 日志

**可能原因**：
- 获取排行榜数据失败
- 用户记录不存在（新用户）

**检查**：
- 查看是否有 `[TEST-10.3] ⚠️ 未找到用户记录` 日志
- 查看是否有 `[TEST-10.3] ❌ 获取当前总分数失败` 日志

### 问题 2：`pkScore` 为 0

**可能原因**：
- PK 对战中答错了所有题目
- 没有完成答题

**解决**：
- 重新进行一局 PK 对战，确保至少答对 1 题

### 问题 3：`newTotalScore` 没有增加

**可能原因**：
- 获取当前总分数失败，使用了 0 作为初始分数
- PK 本局得分为 0

**检查**：
- 查看 `[TEST-10.3] 📊 分数计算` 日志
- 确认 `currentTotalScore` 和 `pkScore` 的值

### 问题 4：上传成功但排行榜分数没更新

**可能原因**：
- 后端返回 `code: 0` 但实际没有更新（新分数 <= 旧分数）
- 排行榜页面没有刷新

**检查**：
- 查看 `[TEST-10.3] ✅ 分数上传成功` 日志中的 `message` 字段
- 如果是 `"未破纪录"`，说明新分数 <= 旧分数
- 确认排行榜页面是否调用了 `loadRankData()`
