# TEST-4.2: AI 助教 Markdown 渲染验证指南（Wise 风格）

## 📋 测试目标
验证 AI 助教返回的 Markdown 内容是否正确渲染为 **Wise 风格**（简洁、白底、清晰），特别是列表和粗体的显示效果。

## ✅ 前置条件
1. 确保网络连接正常（用于调用 AI API）
2. 确保 API Key 配置正确

## 🔍 测试步骤

### 步骤 1: 进入 AI 助教页面
1. 打开微信开发者工具
2. 进入 AI 助教页面 (`pages/chat/index.vue`)
3. 打开控制台（Console）

### 步骤 2: 发送测试消息
1. 在输入框中输入以下测试消息：
   ```
   请用列表总结一下《红楼梦》金陵十二钗中的前三位，并用粗体标出她们的名字。
   ```
2. 点击发送按钮（↑）
3. 观察控制台日志：
   ```
   [Chat] 发送消息: ...
   [Chat] ✅ AI 回复成功，内容长度: X
   [Chat] 📝 AI 回复内容预览: ...
   [Chat] 🎨 Markdown 格式检测: { hasList: true, hasBold: true }
   [Chat] 🎨 开始渲染 Markdown，内容长度: X
   [Chat] 📋 开始无序列表
   [Chat] ✅ Markdown 渲染完成，HTML 长度: X
   ```

### 步骤 3: 验证 UI 显示

#### 列表验证（关键）
- [ ] **列表背景**：应该是**透明/白色**，不是黑色/灰色代码块
- [ ] **列表样式**：显示为普通文本列表（带•点或数字）
- [ ] **列表文字**：深色文字（`#1C1C1E`），清晰可见
- [ ] **列表间距**：行间距适中（`line-height: 1.8`），不拥挤

#### 粗体验证（关键）
- [ ] **粗体文字**：如 **林黛玉** 应该清晰可见
- [ ] **粗体颜色**：深色（`#1C1C1E`），与周围文字对比明显
- [ ] **粗体字重**：`font-weight: 700`，足够粗
- [ ] **粗体背景**：透明背景（`background: transparent`），不是黑色背景

#### 整体排版验证
- [ ] **气泡宽度**：适中，不超过屏幕 72%
- [ ] **字体大小**：30rpx，舒适易读
- [ ] **行间距**：1.8，不拥挤
- [ ] **整体风格**：简洁清晰，符合 Wise.com 风格

#### Loading 状态验证
- [ ] **Loading 动画**：AI 回复生成时，显示三个跳动的点
- [ ] **动画流畅**：无卡顿，动画自然

## 🐛 已知问题排查

### 问题 1: 列表显示为黑色代码块
**可能原因**：
- `rich-text` 组件默认样式覆盖
- CSS 样式未正确应用

**解决方法**：
- 检查 `renderMarkdown` 返回的 HTML
- 确认列表样式包含 `background:transparent`
- 检查是否有全局 CSS 覆盖

### 问题 2: 粗体文字不清晰
**可能原因**：
- `font-weight` 不够粗
- 颜色对比度不够

**解决方法**：
- 确认 `font-weight: 700`
- 确认颜色为 `#1C1C1E`（深色）
- 确认背景为 `transparent`

### 问题 3: 整体排版拥挤
**可能原因**：
- `line-height` 太小
- `margin` 和 `padding` 不足

**解决方法**：
- 确认 `line-height: 1.8`
- 确认列表项有适当的 `margin` 和 `padding`
