# TEST-3.7 清空错题数据方法（最终版）

## ⚠️ 问题说明

在微信开发者工具的控制台中，直接执行 `uni.removeStorageSync()` 可能不可用，因为：
- `uni` 对象可能未全局暴露
- 控制台环境与页面环境不同

## ✅ 推荐方法（按优先级排序）

### 方法 1: 使用 Storage 面板（最简单，强烈推荐）⭐

1. 打开微信开发者工具
2. 点击左侧 **"Storage"** 标签（或 "存储" 标签）
3. 在存储列表中，找到 `mistake_book` 键
4. 右键点击该键 → 选择 **"Delete"**（删除）
5. 或者直接选中后按 `Delete` 键
6. 刷新错题本页面（或重新编译）

**优点**：无需写代码，操作简单直观

---

### 方法 2: 使用 URL 参数（已集成到代码中）⭐

我已经在代码中添加了测试模式，可以通过 URL 参数清空数据：

1. 在微信开发者工具中，点击错题本页面
2. 在地址栏或页面路径后添加参数：`?clear=true`
3. 例如：`/pages/mistake/index?clear=true`
4. 页面加载时会自动清空数据并显示提示

**优点**：无需手动操作 Storage，代码已集成

**使用方式**：
- 在微信开发者工具中，修改页面路径为：`pages/mistake/index?clear=true`
- 或者通过代码跳转：`uni.navigateTo({ url: '/pages/mistake/index?clear=true' })`

---

如果上述方法都不行，可以在 `pages/mistake/index.vue` 的 `onLoad` 方法中临时添加：

---

### 方法 4: 使用 AppData 面板（备用方法）

1. 打开微信开发者工具
2. 点击顶部 **"AppData"** 标签
3. 在数据树中找到 `storage` → `mistake_book`
4. 右键删除或直接修改为空数组 `[]`
