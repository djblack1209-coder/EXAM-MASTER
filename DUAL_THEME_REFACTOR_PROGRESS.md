## 🎨 主题配色参考

### Wise 主题
**浅色模式**:
- 主色: `#00a96d` (绿色)
- 背景: `#FFFFFF` (白色)
- 按钮: 绿色背景 + 白色文字
- 卡片: 白色背景 + 黑色文字
- 导航栏: `rgba(255, 255, 255, 0.8)` 磨砂白玻璃

**深色模式**:
- 主色: `#9FE870` (亮绿色)
- 背景: `#163300` (深绿色)
- 按钮: 亮绿色背景 + 黑色文字
- 卡片: 深绿色背景 + 白色文字
- 导航栏: `rgba(30, 58, 15, 0.8)` 磨砂深绿玻璃

### Bitget Wallet 主题
**深色模式**（默认）:
- 主色: `#0A84FF` (蓝色)
- 背景: `#000000` (纯黑)
- 按钮: 蓝色背景 + 黑色文字
- 卡片: 渐变蓝色 `linear-gradient(135deg, #1a2332 0%, #2d3e50 100%)`
- 导航栏: `rgba(44, 44, 46, 0.8)` 磨砂灰玻璃

## 🔧 技术实现

### 主题切换机制
```javascript
// 使用 Pinia Store
import { useThemeStore } from '@/stores'

const themeStore = useThemeStore()

// 切换主题类型
themeStore.setThemeType('wise') // 或 'bitget'

// 切换深色模式
themeStore.toggleDarkMode()

// 获取当前主题配置
const config = themeStore.currentThemeConfig
```

### 样式应用
```vue
<template>
  <view :class="themeStore.themeClass">
    <!-- 内容 -->
  </view>
</template>

<style lang="scss">
@import '@/styles/theme-wise.scss';
@import '@/styles/theme-bitget.scss';
</style>
```
