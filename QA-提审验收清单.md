## 1. 邮箱注册 / 登录（真邮箱）

### 1.1 密码显示按钮

- [ ] 登录模式密码框右侧有“显示/隐藏”按钮

- [ ] 点“显示”后密码可见，点“隐藏”后恢复密文

### 1.2 首次发送验证码

- [ ] 输入真实新邮箱后，点击“发送验证码”有反应
- [ ] 按钮先显示“发送中...”，随后进入倒计时
- [ ] 邮箱 3 分钟内真实收到验证码

### 1.3 注册成功

- [ ] 输入验证码 + `StrongPass1` 后可成功注册
- [ ] 注册成功后自动进入首页

### 1.4 再次登录

- [ ] 退出登录后，使用刚注册邮箱 + `StrongPass1` 可以重新登录

### 1.5 二次发送验证码

- [ ] 倒计时结束后可再次点击“发送验证码”
- [ ] 第二次验证码邮件能真实收到
